# rock-paper-scissors

playGame runs
humanScore and computerScore variables are initialized at 0

playRound runs, uses getHumanChoice and getComputerChoice functions to set its parameters for the round
determines the round winner by testing humanChoice and computerChoice arguments against each other
round winner is announced in console
humanScore and computerScore are adjusted accordingly
updated scores announced in console

playRound runs again, to a total of five times

determineWinner runs, uses humanScore and computerScore to set its parameters for the game
tests humanFinalScore and computerFinalScore arguments against each other
game winner is announced in console
